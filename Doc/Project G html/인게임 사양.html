E. 몬스터 스탯 상세 사양
116
스탯명설명수식
117
HP몬스터의 체력 0이하로 떨어지면 몬스터 사망Round(Wave.BaseHp * (Monster.HpRate/100))
118
Speed몬스터가 1칸(타일1개의 1개 선분 길이)를 이동하는데 걸리는 시간(s) *100Speed 값 : Round(Wave.BaseSpeed * (Monster.SpeedRate/100))
119
ex) Speed 값이 10인 경우 : 1칸을 0.1초에 통과1칸 이동에 걸리는 실제 시간 : Speed값 /100
120
Resist몬스터가 이동속도 감소, 스턴에 걸렸을때, 실제로 적용되는 비율(%)Round(Wave.BaseResist * (Monster.ResistRate/100))
121
ex) Resist값이 80인 경우 이동 속도 감소, 스턴 적용 시간의 80%만 적용
122
RewardAmount몬스터 처치시 지급되는 재화량Round(Wave.BaseAmount * (Monster.ResistAmount/100))
123
ex) 10일 경우 처치 시마다 재화 10 제공
